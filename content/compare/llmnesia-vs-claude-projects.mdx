## What Claude Projects actually does

Claude Projects is a feature in Claude's paid tiers that lets you create persistent workspaces. Inside a Project, you can:

- Upload files and documents that Claude references across conversations.
- Set custom instructions that apply to every conversation in the project.
- Keep a shared conversation history within the project scope.
- Collaborate with team members on the same project context.

It is an excellent tool for structured, ongoing work — a coding project, a research brief, a long-term client engagement. The value is that Claude remembers the project context without you having to re-explain it every session.

## Where Claude Projects ends

Projects are designed for work you are actively doing. They are not designed for retrospective retrieval — going back weeks or months to find a specific answer, prompt, or decision from any conversation you have had across all your AI tools.

If you ask "what did I decide about the API design three weeks ago, and was it in Claude or ChatGPT?" — Claude Projects cannot answer that. It only knows its own project contents.

## What LLMnesia does differently

LLMnesia is not a workspace organiser. It is a search layer that runs in the background across Claude, ChatGPT, Gemini, and other platforms.

- **Cross-platform search** — one index covers Claude, ChatGPT, Gemini, and more simultaneously.
- **Retrospective retrieval** — search anything from any past session, not just active project files.
- **Jump-back links** — results return you to the original conversation with a direct link.
- **Local-first** — your conversation index stays on your device.
- **Automatic** — no manual file upload or project setup required.

## The overlap and the gap

Both tools help you maintain context over time. The assumptions differ significantly:

**Claude Projects** assumes you know what project work belongs to, you are working in Claude, and you want Claude to maintain that context proactively for ongoing structured work.

**LLMnesia** assumes you may not remember which tool you used or when, you need to search across everything, and retrieval must be fast regardless of platform.

## Practical choice

**Use Claude Projects if:**
- Your team is Claude-centric with clearly-defined ongoing projects.
- You want Claude to maintain shared file context across team members.
- Your retrieval needs are within-project and within-Claude.

**Use LLMnesia if:**
- Your work spans multiple AI tools — Claude, ChatGPT, Gemini, others.
- You need to find past answers that exist outside any Project or predate it.
- You want automatic, cross-platform retrieval with no setup per project.
